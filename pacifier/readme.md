
# 四轴矢量飞行器

具体看视频（暂时还没录好）或自己游戏里打开看。该飞行器只有三个模式，前后平动、配平飞行和悬停，高度和方位实时被动控制，可以在界面上拉条子去修改。

### 使用注意事项

1.这是一个带PyQt的GUI工程，别忘了安装界面相关的库，具体看requirements

2.需要配合载具的name tag才能使用，以便更方便地寻找到对应方位的引擎和转轴

3.由于参数的原因，使用原本代码时必须配合上传的飞行器一同使用，否则需要自己调参

4.部分函数是chatGPT写的，已经在注释中标出，没去仔细看过具体的过程
